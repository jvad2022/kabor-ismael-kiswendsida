I/ AUDIT ORGANISATIONNEL-ASSISTANCE EN GESTION-CONCEPTION DE LOGICIELS

Le terme informatique désigne l’ensemble des techniques et des méthodes de traitement des informations à l’aide d’un ou plusieurs ordinateurs. Notre cabinet a développé depuis janvier 2019, un « logiciel d’application » de gestion de stock du nom de « Cadora G.Stocks » dans le but d’accompagner ses clients. Ce logiciel est simple et pratique pour tous utilisateur, la base de données (tables, formulaires, requêtes) a été conçu sous Microsoft Access. Les classes et modules ont été programmés avec le langage SQL (Structure Query Langage) ce qui rend la base de données très puissante. L’interface principale se décompose comme suit : - Bouton Nouvelle commande clients (facturation des ventes) - Bouton Nouveau Bon de commande (commandes fournisseurs) - Mise a jour automatique du stock - Les commandes actives sont affichées à l’écran - Inventaire à réapprovisionner - Liens rapides concernent les états de contrôle de stock. Un fichier audit permet aux responsables de connaitre toutes les actions des utilisateurs dans le logiciel en temps réel (nom utilisateur, action, date et heure).

Ecran de connexion logiciel de gestion de stocks CADORA version 1.0 bureau et android

Ce logiciel est utilisé en monoposte et réseau avec la possibilité d'extension en Android utilisable sur les téléphones smartphone. NB : en cas d’absence de logiciel de gestion de stock dans votre structure, un contrat d’assistance en gestion passé avec notre cabinet, vous donne droit à l’installation du dit logiciel pour permettre aux consultants de mener à bien la mission confié.

Interface de travail du logiciel. Entrez votre nom pour créer un nouveau utilisateur, pareil pour créer un nouveau client, produit, fournisseur. Cliquez sur commande client pour les ventes et bon de commande pour les achats.

Nos conseils sur l’installation de PYCHARM et Visual Studio Code : • PYCHARM n’existe pas en français son interface est en anglais, par contre Visual Studio Code vous permet de faire une extension en installant le french pour avoir l’interface en français. Nous vous déconseillons fortement d’utilisé l’interface en français. L’interface en anglais vous permettra au passage d’apprendre quelques mots en anglais qui sont très utiles dans le monde de la programmation. • Souvent quand vous commencé à tapez une commande avec le terminal de PYCHARM, il vous propose de compléter automatiquement cette commande pour vous. Pour éviter les erreurs de frappe, nous vous recommandons de désactiver cette fonction dans les settings. Les commandes on les connait, le but c’est de les apprendre et non les complétées automatiquement.
